Make SQL database management, preprocessing, and exploratory analysis faster and safer.

Connect via SQLAlchemy, validate with a test query, cache the connection for performance, and browse tables instantly.

Key Functionalities

Everything you listed ‚Äî organized like a product page.

Database Connection

SQL Server connection + caching

Uses SQLAlchemy, validates with SELECT 1, and caches the connection using st.cache_resource for speed.

Table Preview & Stats

Preview rows + column analytics

See top rows, total rows/cols, null counts, and column details (dtype, null/non-null counts) in one place.

Data Type Conversion

Safe dtype changes with warnings

Supports special handling like DATE ‚Üí INT (YYYYMMDD). Risky conversions require confirmation before applying.

Null Handling

Smart strategies by data type

Numeric: mean/median/mode/interpolation/ffill/bfill/custom. Datetime: interpolation/ffill/bfill/custom. Text: mode/custom/Unknown/N/A.

Duplicate Removal

CTE + ROW_NUMBER cleanup

Detects duplicates across all columns and removes them safely using a CTE approach in SQL Server.

Natural Language SQL

LLM-powered SELECT queries

LangChain + Ollama translates English into SQL, runs it, and shows results interactively. Non-SELECT queries are blocked for safety.

Interactive Joins

Join tables visually

INNER/LEFT/RIGHT/FULL OUTER joins, selectable join keys, optional WHERE clause, column picker, and preview before export.

Export

CSV download or save-to-SQL

Export processed/joined data as CSV or write back as a new SQL Server table, with validation for safe names.

UI Polish

Modern Streamlit styling

Custom gradients, animated headings, styled metrics/tables/alerts, progress feedback, and smooth UX patterns.

Workflow Summary

A simple, guided flow from connection ‚Üí cleaning ‚Üí analysis ‚Üí export.

1

Connect to Database

2

Select Table + Preview

Pick a table, view the first 10 rows, and review null counts + column stats.

3

Change Data Types

Convert types safely with guardrails, warnings, and success/skipped/failed feedback.

4

Handle Null Values

Apply per-column strategies (including interpolation) or delete rows with nulls at once.

5

Remove Duplicates

6

Query + Join + Export

Run SELECT queries via natural language, join tables interactively, export to CSV or save as a new SQL table.
